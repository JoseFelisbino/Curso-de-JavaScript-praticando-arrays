# Faça esse curso de JavaScript e:

- Revise o conceito de array em lógica de programação.

- Pratique como acessar arrays em JavaScript e manipular dados.

- Relembre o uso do laço for para acessar a manipular arrays.

- Entenda as diferenças entre for e for...of e quando utilizá-los.

- Revise alguns métodos de array mais usados e como usar a sintaxe de callback.

- Pratique a resolução de exercícios de lógica de programação com arrays.
